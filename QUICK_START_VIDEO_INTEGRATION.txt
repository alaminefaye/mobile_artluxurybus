     • Statut actif/inactif

   cd gestion-compagny
   php artisan serve

ÉTAPE 2 : Lancer l'app Flutter
   cd mobile_dev/artluxurybus
   flutter pub get
   flutter run

ÉTAPE 3 : Tester dans l'app
   1. Se connecter avec un compte admin
   2. Aller dans l'onglet "Services" (bas de l'écran)
   3. Cliquer sur "Gestion des Vidéos"
   4. Tester les fonctionnalités !
